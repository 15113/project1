# Portfolio - Anthony Gallagher

A professional portfolio website for Anthony Gallagher, Software Architect at Carnegie Mellon University.

## Description

This repository contains the source code for a responsive personal portfolio website. It showcases Anthony Gallagher's professional background, core competencies, and notable projects. The site is designed to provide a clean and professional overview of his expertise in software architecture, legacy modernization, and technical due diligence.

## Features

- **Responsive Design:** Built with Bootstrap 5 to ensure a seamless experience across mobile, tablet, and desktop devices.
- **Project Showcase:** Highlights key projects with descriptions and links to source code.
- **About Section:** Detailed information on professional experience, skills, and areas of expertise.
- **Contact Information:** Easy access to professional profiles on GitHub and LinkedIn.

## Technologies Used

- **HTML5 & CSS3:** For structure and custom styling.
- **Bootstrap 5:** For responsive layout and UI components.
- **jQuery:** For enhanced interactivity.
- **JavaScript:** For custom scripting and dynamic behavior.

## Project Structure

- `index.html`: The main entry point for the portfolio.
- `assets/css/`: Contains custom CSS stylesheets.
- `assets/images/`: Stores profile pictures and project screenshots.
- `assets/js/`: Contains JavaScript files for site functionality.

## Getting Started

To view the portfolio locally:
1. Clone the repository.
2. Open `index.html` in any modern web browser.
